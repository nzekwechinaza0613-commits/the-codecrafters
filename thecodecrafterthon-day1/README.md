# README

## HOW TO USE THE CLI CALCULATOR

     The calculator handels basic operations like addition(+), subtraction(-), multiplication(*), and division(/). 
     The calculator askes for the user's first value afterwards asks for the intended operator(+,-,*,/), the second value before the result is displayed on the terminal. 

    Whenever the user finds any operation difficult he/she easily types 'help' and a list of the available comand is shown to aid . The program keeps running until the user types 'quit' then it leaves with an exit message.

    if a wrong operation is carried out the program print an error message and allows the user to keep trying again and again.
